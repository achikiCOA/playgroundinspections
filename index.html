Get Started

Use this app to complete playground inspections, save inspection history, and track repair items on the phone or tablet being used.

  1. 1
    Choose the playground.
    Select the correct playground card. Add or edit cards before inspections if your site has multiple play areas.
  2. 2
    Fill in the inspection details.
    Enter the inspector name, confirm the date and time, and add weather or surface conditions.
  3. 3
    Complete every checklist item.
    Use Pass, Issue, or N/A. For a clean inspection, tap Mark All Pass, then review anything that should be changed.
  4. 4
    Document issues clearly.
    Choose the priority, add repair notes, set the repair status, and attach a useful photo when needed.
  5. 5
    Save and review repairs.
    Saving resets the form for the next inspection. Open Repairs to update work order status.
  6. 6
    Back up records.
    Records stay on this device. Use Export JSON for backups or transfer, Import JSON on another device, and CSV for spreadsheet review.

Inspection Guidance

Reference prompts
Priority scale 1: correct immediately. 2: correct as soon as possible. 3: correct when time permits. 4: monitor for worsening.
Storage note Saved inspections stay on the device and browser that entered them. Use Export JSON and Import JSON to move records between devices, or connect a cloud database for automatic sync.
Ohio child care notes Quarterly playground inspection records should be kept for one year. Outdoor spaces need adequate usable area, protected boundaries, functional gate latches, shade, bathroom/drinking-water access, and safe fall zones.
CPSI / NRPA focus Surface protection, accessibility, supervision, age-appropriate equipment, use zones, protrusions, entrapment, trip hazards, crush/shear points, and routine maintenance.
Sources NRPA Daily Dozen, Ohio Admin. Code 5180:2-12-11, Frequent Inspection Form
